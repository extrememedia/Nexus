# Changelog

All notable changes to Nexus Auto Trader will be documented in this file.

## [1.0.0] - 2025-01-28

### Added
- Initial release
- 500+ cryptocurrency support via CoinGecko API
- 150+ stocks with sector filtering
- Fear & Greed sentiment analysis
- RSI and EMA technical indicators
- 5 trading strategies (Conservative, Moderate, Aggressive, YOLO, Scalper)
- Real-time price updates every 2 seconds
- Paper trading mode with $100 starting balance
- Live trading support (Coinbase, Alpaca)
- Telegram alert integration
- Advanced filtering (price, change, sector, search)
- Manual quick-buy buttons
- Position management with dynamic take profit/stop loss
- Responsive dark theme UI

### Features
- Contrarian trading based on market sentiment
- Momentum detection and volume spike alerts
- Market trend analysis (bullish/bearish/neutral)
- Win rate tracking and trade history
- Sortable asset lists
- Asset detail view with buy functionality
